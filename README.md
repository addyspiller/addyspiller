## Client work

**Building Decarbonization Platform**
Multi-step workflow guiding building owners through energy efficiency
planning and property analysis. Integrated ArcGIS for location data.
Built for investor and partner walkthroughs and feedback.
`React · TypeScript · Vite · Netlify`

**AI Insurance Policy Comparison Tool**
Frontend for a fintech SaaS that uses Claude API to analyze insurance
renewal PDFs, classify changes by risk level, and generate plain-language
explanations for SMB owners. Secure direct-to-S3 PDF upload, Clerk auth,
result history. Built to validate functionality and gather prospective
client feedback.
`Next.js · TypeScript · Shadcn/UI · Vercel`

**AI-Powered Zoning & Feasibility SaaS**
Full prototype for a B2B real estate analytics platform — landing page,
auth flow, mock Stripe checkout, and a dashboard with zoning compatibility
analysis, environmental risk assessment (FEMA + NWI data), and PDF report
export. Built to gather prospective client input and as a dev team handoff
spec. Also built the public-facing marketing site to communicate the platform's vision while the product is in development.
HTML · CSS · React · TypeScript · Supabase · Recharts · Vercel

**Support Intake Form**
Customer support tool for a recycling platform. Progressive disclosure
design — instant answers first, contact forms only when needed. Smart
routing to the right team member based on issue type. Zip-based location
search, photo upload to better support service issues, B2B partnership inquiry flow. Built
for internal alignment and approaching dev handoff.
`TypeScript · React · Netlify`

**Impact Research Platform**
Analytics dashboard with integrated TEA and LCA calculators, industry
analytics, and client scheduling flow. Built for client feedback and dev
handoff.
`React · TypeScript · Supabase`

**Professional Development Community Platform**
Prototype supporting a dual B2B/B2C product strategy pivot. Built for internal alignment and dev handoff.
`React · TypeScript`

**Neighborhood Community Platform**
Public website and authenticated admin dashboard built to replace an
existing site for a NYC block association. Member management, event
RSVPs, and dues collection.
`Next.js · Supabase · Stripe · Vercel`

## Personal builds

**Pret-à-Parler** _(private)_
A French language learning app built for one user — my dad — targeting
C1 proficiency. Real-time pitch detection and pronunciation feedback,
YouTube transcript integration for cultural immersion, and on-device ML
via HuggingFace Transformers. He opened it to his French class for
broader feedback — a few real users, real validation.
`TypeScript · React · HuggingFace · WaveSurfer.js · Netlify`

**NYC Recycle Smart** _(private)_
Snap a photo of any item, get instant NYC-specific disposal instructions.
Built after learning that 33% of NYC trash is recyclable but ends up in
landfill. Constructed in 4 days.
`JavaScript · React · Google Vision · Netlify`

**[Aperture](https://github.com/addyspiller/aperture-readme)**
A privacy-first behavioral pattern interpreter. Surfaces insights from
transaction and location data without judgment, scoring, or optimization.
Raw coordinates discarded at parse time. Active development.
`Python · FastAPI · PostgreSQL · React · TypeScript · Google OAuth · Netlify`
